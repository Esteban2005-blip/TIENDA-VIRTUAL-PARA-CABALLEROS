# Tienda de Ropa para Caballeros

Este proyecto es un sitio web completo para una tienda de ropa masculina, diseñado con enfoque en experiencia de usuario, accesibilidad y mejora continua.

## Estructura del sitio
- **Inicio**: Banner, frase, productos destacados, botón "Comprar ahora".
- **Catálogo**: Lista de camisas, pantalones y chaquetas, cada uno con imagen, nombre, precio y botón "Agregar al carrito".
- **Carrito**: Productos seleccionados, cantidad, precio total, botón "Finalizar compra" y opción de eliminar productos.
- **Formulario de compra**: Nombre, dirección, teléfono, método de pago simulado, validación de campos.
- **Contacto**: Correo, redes sociales, ubicación.

## Funcionalidades extra
- Buscador de productos.
- Filtro por tipo de ropa.
- Botón de favoritos.

## Diseño
- Navegación intuitiva (menú visible en todas las páginas).
- Diseño responsivo (PC y móvil).
- Accesibilidad (letras legibles, buen contraste, botones grandes).
- Feedback al usuario (ej: mensaje "Producto agregado al carrito").

## Evaluación y mejoras
- Preparado para pruebas de usuario y documentación de informe.
- Estructura lista para aplicar heurísticas de Nielsen y registrar mejoras.
